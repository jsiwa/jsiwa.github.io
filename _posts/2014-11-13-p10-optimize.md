<h4>1. 缓存for循环的数组</h4>
<pre><code class="javascript">
BAD: 
for (var i = 0; i < myArray.length; i++){
	// do stuff
}
BETTER: 
var len = myArray.length;
for (var i = 0; i < len; i++) {
	// do stuff
}

for (var i = 0, len = myArray.length; i < len; i++){
	// do stuff
} 

var len = myArray.length;
while (len--){
	// do stuff
}
</code></pre>
